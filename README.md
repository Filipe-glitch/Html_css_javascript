# 📚 Estudos em HTML, CSS e JavaScript

Repositório dedicado aos meus estudos em desenvolvimento web, utilizando **HTML, CSS e JavaScript**, com foco em conceitos teóricos e práticos.

## 🚀 Objetivo

Aprender e consolidar os fundamentos do desenvolvimento front-end.

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 📚 Conteúdos estudados

### 🔹 HTML
- Estruturação de páginas web
- Uso de tags semânticas e básicas
- Criação de formulários e elementos interativos
- Organização do conteúdo

### 🔹 CSS
- Estilização de páginas
- Controle de layout (Flexbox e Grid)
- Animações e transições
- Personalização de botões e elementos visuais
- Ajustes de espaçamento e organização

### 🔹 JavaScript
- Fundamentos da linguagem
- Manipulação de elementos da página (DOM)
- Criação de interações simples
- Validação de dados
- Uso de funções para organização do código

## 💡 Observações

- Repositório focado em aprendizado contínuo
- Contém exemplos práticos e exercícios
- Base para projetos futuros mais completos

## 📌 Aprendizados

- Fundamentos do desenvolvimento web
- Integração entre HTML, CSS e JavaScript
- Criação de interfaces básicas
- Lógica aplicada ao front-end
